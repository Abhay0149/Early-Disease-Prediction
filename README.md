# 🩺 Early Disease Prediction using Machine Learning

## 📌 Overview
This project presents an end-to-end machine learning pipeline for predicting diabetes risk using the Pima Indians Diabetes Dataset. Multiple machine learning models, hyperparameter tuning, ensemble learning, and explainable AI techniques were used to build an accurate and interpretable prediction system.

---

## 🎯 Objective
- Predict the risk of diabetes using clinical parameters.
- Compare multiple machine learning algorithms.
- Improve performance using hyperparameter tuning and ensemble learning.
- Provide model explainability using SHAP.
- Demonstrate practical healthcare AI applications.

---

## 📊 Dataset
**Pima Indians Diabetes Dataset (UCI Machine Learning Repository)**

- Total Records: 768
- Original Features: 8
- Target Variable:
  - 0 → Non-Diabetic
  - 1 → Diabetic
